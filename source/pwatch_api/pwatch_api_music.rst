=========================
可编程音乐模块
=========================

SmartWatch具有可编程音乐模块

播放预置音乐
=========================
pwatch.playMusic(name)
 播放名称name的预置音乐

====================  ===============  
   name                  音乐描述              
====================  ===============  
Fido                       提示音              
Tiantic                    我心永恒       
20thCenFox                 电影配乐              
HBday                      生日快乐歌           
MissionImp                 007音乐           
Picaxe                     友谊地久天长            
smb                        超级玛丽          
Urgent                     紧急音                         
====================  ===============


播放音符
=========================
pwatch.playTone(freq,beat)
 freq为音符频率,170-3600，beat为节拍,1/16-16
 

设置演奏速度
=========================
pwatch.setToneBpm(value)
 value为速度 30-200
